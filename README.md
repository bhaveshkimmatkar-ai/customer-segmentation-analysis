# Customer Segmentation Analysis

## Tools Used
Python | Pandas | K-Means Clustering | RFM Analysis

## About
Segmented customers into behavioral groups based on
purchasing patterns using RFM Analysis and K-Means Clustering.

## Key Insights
- Customers classified into high-value, at-risk, and new segments
- RFM Analysis used for recency, frequency, monetary scoring
- Actionable insights generated for targeted marketing campaigns
- Data visualizations created for stakeholder decision-making
